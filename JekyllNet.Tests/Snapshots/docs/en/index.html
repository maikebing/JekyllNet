About

GitHub Pages · Jekyll-compatible · .NET 10

A .NET static site generator moving toward GitHub Pages behavior

Jekyll.Net is built with C# and .NET 10 and is gradually filling in the core pieces most project documentation sites need: Front Matter, Layouts, Includes, Collections, Tags, Categories, and Sass.

In one sentence

This project is not wrapping Jekyll itself. It is rebuilding a GitHub-Pages-style static generation path in C# piece by piece.

Runtime .NET 10

What already works

The repository already covers the core pieces needed by a small GitHub Pages style documentation site.

Content parsing

Supports _config.yml, YAML Front Matter, Markdown to HTML, and a basic set of tags and filters.

Templating

Supports _layouts, nested layouts, _includes, _data, and output-enabled collections.

Site assets

Supports Sass or SCSS compilation and copies static assets into the _site output directory.
