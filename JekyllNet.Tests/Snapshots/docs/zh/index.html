GitHub Pages · Jekyll-compatible · .NET 10

用 .NET 写一个越来越像 GitHub Pages 的静态站点生成器

Jekyll.Net 以 C# 和 .NET 10 为基础，正在补齐 GitHub Pages 常见工作流需要的 Front Matter、Layout、Include、Collection、Tag、Category 和 Sass 能力。

开始阅读 查看 sample-site 阅读 README
  • .NET 10
  • GitHub Pages
  • Jekyll-style
  • Markdown + Sass

Quick Start

dotnet run --project .\JekyllNet.Cli -- build --source .\sample-site

一句话理解

这个项目不是在包装现成的 Jekyll，而是在用 C# 逐步重建一条能服务 GitHub Pages 风格站点的静态生成链路。

运行时 .NET 10
当前入口 CLI build
站点特性 Markdown + Liquid 子集
目标方向 靠近 GitHub Pages

当前已经具备什么

仓库当前实现已经覆盖一个 GitHub Pages 风格站点最核心的一批静态生成能力。

内容解析

支持 _config.yml、YAML Front Matter、Markdown 到 HTML，以及基础的标签和过滤器。

模版能力

支持 _layouts、嵌套 layout、_includes、_data，以及带输出的 collections。

站点资源

支持 Sass/SCSS 编译，并把静态资源复制到 _site 输出目录。
