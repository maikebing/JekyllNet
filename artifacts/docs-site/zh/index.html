.NET 静态站点生成

吾以 C# 造 Jekyll 风格之站

吾名 JekyllNet，以 C# 与 .NET 10 为器，务在补齐文档站与内容站常用之道：Front Matter、布局与 Include、Collections、文章、分页、Sass、多语文档，以及诸多贴近 GitHub Pages 之行为。

览文档导航 观 sample-site 读首发新闻
  • .NET 10
  • GitHub Pages 风格
  • Markdown + Liquid
  • CLI build/watch/serve

起步二令

dotnet run --project .\JekyllNet.Cli -- build --source .\sample-site
dotnet run --project .\JekyllNet.Cli -- serve --source .\docs --port 5055

一言以蔽之

吾非包裹既有 Jekyll 者，乃欲以 .NET 重铸一条近乎 GitHub Pages 心法之生成路径。

今已可为何事

吾之实现，今已足以支撑真实文档站、小型内容站，以及主题兼容之验证。

内容之道

已通 YAML Front Matter、Markdown、Collections、Posts、Tags、Categories、摘要、Draft/Future/Unpublished 诸语义。

主题之道

已通嵌套 Layout、Include、常见 Liquid 控制语法、高价值 Filters、defaults、静态文件 Front Matter 与 Sass。

工程之道

CLI、Snapshot 回归、GitHub Actions 示例、dotnet tool 打包元数据与 winget 模板，今皆在库中。
