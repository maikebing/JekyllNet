# About This Site

This site directly borrows the outer documentation shell from the `LVGLSharp` `docs` folder.

## What was kept

- the layout organization
- the top navigation and footer structure
- the main CSS visual style
- the panel-based landing page pattern

## What was not copied

- the original repository content
- the original project links and branding
- the original analytics script
- the original information architecture tied to LVGLSharp itself

## What this site is for now

It gives Jekyll.Net a clear, lightweight, GitHub Pages friendly documentation site and a good place to keep expanding project guidance over time.
