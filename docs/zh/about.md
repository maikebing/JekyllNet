# 关于本站

此 `docs` 目录，今已非占位之页，乃吾正式之门面与手册也。

## 何以分三层

吾故分其站为三：

- **文档**：主导航、起步、兼容边界与部署之法
- **博客**：特性、配置、CLI、多语与 AI 翻译之详解
- **新闻**：里程碑与版本动态之记

如此，则入口不繁，而长文亦得其所安。

## 何以于技术亦有益

此站本身，即吾之真实 fixture，常用以覆盖：

- 中英双语路由与 translation links
- layout、include、导航与页脚
- `_config.yml` defaults 与多语 UI 文辞
- 基于 fixture 的构建回归测试所护之生成结果
