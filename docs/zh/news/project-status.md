# docs 站点已落地

当前仓库已经补上一套 `docs` 目录形式的 GitHub Pages 文档站。

## 这次落地的重点

- 直接复用了参考站点的页面壳和 CSS 风格。
- 没有照搬参考站点的正文内容。
- 新内容围绕 `Jekyll.Net` 当前能力、快速开始和 Pages 配置展开。
- 中英文入口保持了同构，后续追加页面时更容易同步。

## 接下来最自然的演进方向

可以继续往这套站点里补：

- 更多与 GitHub Pages 兼容性相关的专题页
- 当前 Liquid 子集实现细节
- 更多 sample-site 示例拆解
